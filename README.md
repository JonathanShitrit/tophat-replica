# tophat-replica
A replica of TopHat for our CS 355 course

visit: https://indelible-interactive.herokuapp.com/

### Functionality
Account

* Login 

* Registration

Instructor

* Create Questions

* Create Assignment (Problem Set or Exam) from Questions

* View My Students' Scores

Student 

* Complete Assignment

* View My Own Scores 

Support 

* About Developers

* Contact Us

* Help Page (HTML or video)
